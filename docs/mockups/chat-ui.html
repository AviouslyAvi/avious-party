Toast sits just above the composer so it never pushes chat history out of view. Cross-domain switches are dropped silently (one system message in chat instead of a navigation). Detection logic per site: YouTube hooks yt-navigate-finish; Vimeo/generic watch <video src>; Netflix polls location.pathname.

Let your browser ping you?

Push notifications fire for every new chat message, even when this tab is focused. We only ask once — you can flip it back off anytime.

Lazy prompt: we don't ask for permission on page load. The user flips the toggle, we explain why and call Notification.requestPermission() only when they click Allow. If they decline, the toggle flips back off and stays off until they re-enable.

Popup is light-theme by default (Cotton ground, Terracotta primary) because Chrome's toolbar surface reads as a daytime context — keeping it light against the darker sidebar makes the two surfaces feel distinct. "Start watch party" disables when a room is already active. Master kill is a green Sage toggle at the bottom: it's the only switch the popup is allowed to flip globally.
