🎬 Watch-Party

Sync play, pause & seek for everyone watching the same page. Works on any site with a video player. Chat included. No accounts.

1 Install the userscript

You'll need Tampermonkey (free, Chrome / Firefox / Safari / Edge). Then click:

Install Watch-Party

Tampermonkey will prompt you to confirm. You only do this once.

2 Start a watch party

3 Joining someone else's party?

Just open the link they sent you. The userscript activates automatically — no extra step.

Open source · GitHub · Report a bug

Watch-Party is an independent project. Works on any site that exposes a standard HTML5 video element.
